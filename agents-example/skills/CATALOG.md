# ~/.agents/skills/CATALOG.md

Skills are markdown files injected into the system prompt when loaded.
Place .md files in ~/.agents/skills/ — reference them in config.json under load.skills.

## Example Skill Format

Each skill file should contain:
- What the skill does
- When to apply it
- Any specific instructions or constraints

## Built-in Skills

Add your skill files here and document them in this catalog.
